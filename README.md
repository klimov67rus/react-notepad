# Тестовое приложение Записной книжки

## Использованные технологии

- React
- IndexedDB (Dexie)
- Markdown

## Функционал

- Создание заметок поддерживающих разметку Markdown
- Автоматическое сохранение записей в indexedDB браузера
- Поиск заметок по названию
